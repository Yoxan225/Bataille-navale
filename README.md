Bataille Navale Game 

Une implémentation moderne du célèbre jeu Bataille Navale
🎯 Objectif du projet

Ce projet a été conçu pour :
Implémenter la logique complète d’un jeu stratégique
Créer une interface utilisateur moderne et dynamique

🧠 Fonctionnalités

✔️ Placement des navires
✔️ Attaques tour par tour
✔️ Détection des coups (touché / manqué / coulé)
✔️ Gestion de la grille de jeu
✔️ Interface utilisateur interactive avec React

⚙️ Technologies utilisées
 
Python 
React 

 Installation & Lancement
 Cloner le projet
cd Bataille_navale
npm install
npm run dev
🕹️ Utilisation
Ouvrir l’application dans le navigateur
Placer tes navires sur la grille
Lancer la partie
Attaquer les cases de l’adversaire
Gagner en coulant tous les navires ennemis 🚀
